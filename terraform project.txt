Using terraform create a another instance create and check instance

1. Create a ubuntu server
i. launch instance and give name ubuntu.
ii.select ubuntu and t2.micro cpu 
iii.download a new keypair.
iv. select default vpc. 
v. In security group allow ssh,https and http allow anywhere.
vi. launch instance.

2. Log in to the the ubuntu server using putty
i.copy the public IPv4 and paste on the putty.
ii. Select ssh and go to the authentication and upload the ppk file 
iii. Then put password ubuntu and log in.

3. Update the server
i. use to update server use command 
ii. sudo apt update -y 

4. Install the terraform on the server.
i.visit https://www.terraform.io/ and click the download button
ii. select linux amd64 and cpoy the link address
iii.paste on the server wget https://releases.hashicorp.com/terraform/1.3.4/terraform_1.3.4_linux_amd64.zip
iv.sudo apt install unzip -y
v. sudo unzip terraform_1.3.4_linux_amd64.zip
vi. sudo cp terraform /bin
vii. terraform -v

5. Select registry and create s3 bucket  
i. Go to the home page and click on the registry
ii. Then select the aws provider and go to the documention 
iii. search s3 and copy the resource s3 bucket 
iv. and the provider,region,access_key, secret_key 
v. create folder sudo mkdir s3 and go to the folder cd s3
vi. sudo nano s3.tf and paste the credentional 
vii. sudo terrform init
viii. sudo terrafrom apply
ix. check the output
